# Codename: karucha

Un programa sencillo para hacer avatares en huayra.

## ¿Cómo ejecutarlo?

En GNU/Linux podrías ejecutar:

    git clone https://github.com/hugoruscitti/karucha.git
    make test_linux

O bien en MAC-OSX:

    git clone https://github.com/hugoruscitti/karucha.git
    make test_mac
